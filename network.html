คำถามที่พบบ่อย
คำถาม : งาน Sustainable Event คืออะไร และต่างจากงานอีเวนต์ทั่วไปอย่างไร?
คำตอบ : งาน Sustainable Event คือการจัดงานที่คำนึงถึงผลกระทบต่อสิ่งแวดล้อม สังคม และการใช้ทรัพยากรอย่างคุ้มค่า ต่างจากงานทั่วไปที่มักเน้นความสำเร็จของงานเป็นหลัก 🌱
คำถาม : จัดงาน Sustainable Event ที่ IMPACT ได้อะไรบ้าง
คำตอบ : ได้เอกสารขอบคุณการเข้าร่วมงาน รายงาน Carbon Footprint ได้ลดก๊าซเรือนกระจกจากการจัดงาน สนับสนุนชุมชน และส่งมอบโลกที่ดีต่อไปให้แก่รุ่นหลัง
